# 🤖 AI Interview Preparation Platform

An AI-powered interview preparation platform built using **Spring Boot**, **Thymeleaf**, and **MySQL** to help students prepare for technical interviews through structured learning, company-wise preparation, and AI-powered features.

---

## 🚀 Features

- 🎯 Modern Landing Page
- 👨‍🎓 Student Login (Coming Soon)
- 👨‍💼 Admin Login (Coming Soon)
- 📚 Topic-wise Interview Questions
- 🏢 Company-wise Preparation
- 📊 Student Progress Dashboard
- 🤖 AI Mock Interview (Planned)
- 📄 AI Resume Review (Planned)
